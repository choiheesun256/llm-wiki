# Network Medicine Target Discovery

> 📄 [*bioRxiv* (2025)](https://doi.org/10.1101/2025.05.29.656813)

## 요약

Single-cell transcriptomics, bulk multi-omics, genome-wide CRISPR 스크린, PPI 네트워크를 통합하여 질병 특이적 치료 타겟을 체계적으로 우선순위화하는 network medicine 기반 ML 프레임워크다.

## 핵심 내용

- 다중 데이터 모달리티를 네트워크 위에서 통합하는 ML 프레임워크
- Clear cell renal cell carcinoma에 적용하여 기존 타겟 재발견 및 5개의 새로운 치료 후보 예측
- ENO2 억제가 가장 강한 항종양 효과를 보임

## Drug repurposing과의 관련성

- Multi-omics 통합이 disease module 기반 타겟 발견의 정확도를 높일 수 있음을 시연
- 식별된 타겟에 대해 기존 약물을 재배치하는 다음 단계로 자연스럽게 연결
- Single-cell 데이터 활용으로 세포 유형별 타겟 특이성 확보 가능

## 한계점

- 아직 프리프린트 단계로 동료 검토 미완료
- 특정 암종에 대한 사례 연구이므로 일반화 가능성은 추가 검증 필요
