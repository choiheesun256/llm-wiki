# Phosphoproteomics Bioinformatics Guide

## Summary

Comprehensive review of bioinformatics resources for phosphoproteomics data interpretation. Surveys 16 kinase/phosphatase knowledge bases, phosphorylation site databases, kinase inhibitor databases, and kinase activity inference tools. Includes benchmarking comparisons and usability assessment from biologist and clinician perspectives.

## Key points

- 16 kinase/phosphatase knowledge bases cataloged
- PhosphoSitePlus and Swiss-Prot most actively maintained
- Kinase activity inference tools compared
- Practical guide for tool selection

## Project relevance

- Reference for phosphoproteomics analysis pipeline
- Tool selection guide for kinase signaling studies
- Foundational resource for proteogenomics workflows

## Caveats

- Published 2020; many new tools since then
- Limited benchmarking scope
- Human-focused only
