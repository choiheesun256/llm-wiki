# 3DisoGalaxy: Breast Cancer Isoform Foldome Atlas

## Summary

3DisoGalaxy is an isoform-resolved breast cancer knowledge base integrating PacBio Iso-Seq (n=35), short-read RNA-seq (4 cohorts), and Ribo-seq (n=42). Identifies 123,395 transcript variants and 73,715 ORFs, with 46,601 AlphaFold-predicted protein structures forming a breast cancer foldome. Demonstrates KRAS4A selective motif loss and a non-canonical AKT1 isoform with TNBC-biased expression. Interactive web portal with 3D structure visualization.

## Key points

- 123,395 transcript variants, 73,715 ORFs, 46,601 protein structures
- Long-read (PacBio) + short-read + Ribo-seq integration for isoform resolution
- AlphaFold foldome with structural similarity network for function inference
- KRAS4A and AKT1 non-canonical isoform functional hypotheses

## Project relevance

- Isoform-level analysis beyond gene-level in cancer proteogenomics
- AlphaFold-based structural atlas for cancer-relevant proteoforms
- Drug target implications from isoform-specific structural differences

## Caveats

- Preprint (bioRxiv 2026); not yet peer-reviewed
- Breast cancer-specific; extension to other cancers needed
- AlphaFold predictions, not experimental structures
- Functional validation gap between structure prediction and biology
