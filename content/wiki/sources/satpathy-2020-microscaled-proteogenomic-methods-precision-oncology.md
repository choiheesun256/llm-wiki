# Microscaled Proteogenomics (MiProt)

## Summary

Develops Microscaled Proteomics (MiProt) and Biopsy Trifecta Extraction (BioTExt) protocols enabling proteogenomic profiling from core needle biopsies using only 25μg of peptide input. Achieves >10,000 proteins and >25,000 phosphosites per sample. Applied to ERBB2+ breast cancer PDX models to identify trastuzumab response predictors and resistance mechanisms.

## Key points

- BioTExt extracts DNA, RNA, protein from single 14G core biopsy
- 25μg input yields CPTAC-comparable proteomics depth
- ERBB2/mTOR phosphosite suppression predicts pathological complete response
- Resistance: AR signaling, mucin overexpression, inactive TME

## Project relevance

- Enabling technology for clinical proteogenomics
- Biopsy-scale precision oncology
- Treatment response prediction methodology

## Caveats

- PDX proof-of-concept; large patient cohort validation needed
- Technical complexity for routine clinical labs
- Breast cancer only; other cancer types untested
