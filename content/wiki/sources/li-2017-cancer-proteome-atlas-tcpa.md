# The Cancer Proteome Atlas (TCPA)

## Summary

TCPA (tcpaportal.org) is an open-access bioinformatic resource for RPPA-based cancer proteomic data. Contains ~300 protein markers across ~8,000 TCGA tumor samples (32 cancer types), 500+ independent samples, and 650+ cancer cell lines. Provides interactive analysis modules including correlation, differential expression, survival, drug-protein associations, network visualization, and heatmaps.

## Key points

- ~8,000 TCGA samples across 32 cancer types with RPPA data
- Interactive web portal with multiple analysis modules
- Drug-protein association analysis capability
- 650+ cancer cell lines with drug screening data

## Project relevance

- RPPA data resource for pan-cancer protein expression analysis
- Drug sensitivity-protein expression correlation platform
- Complementary to MS-based proteomics resources

## Caveats

- Limited to ~300 RPPA protein markers (not whole proteome)
- Antibody-dependent platform
- Published 2017; MS-based proteomics now provides deeper coverage
- TCGA data lacks comprehensive treatment/outcome information
