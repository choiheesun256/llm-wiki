# Reactome Pathway Knowledgebase 2024

## Summary

2024 update of the Reactome pathway knowledgebase, a manually curated database of human biological pathways covering ~2,700 pathways and >15,000 reactions. Features expanded disease pathway coverage, improved visualization tools, enhanced API access, and updated drug-target annotations. Essential reference resource for omics data interpretation and pathway enrichment analysis.

## Key points

- ~2,700 curated human pathways, >15,000 reactions
- Expanded disease pathway coverage
- Improved visualization and API access
- Drug-target pathway annotations updated
- Multi-species ortholog inference

## Project relevance

- Reference database for pathway enrichment analysis
- Drug-target pathway mapping resource
- Essential tool for omics data interpretation

## Caveats

- Database update paper; not a methodology study
- Curation bias toward well-studied pathways
- Static pathway representation; no dynamic activity modeling
