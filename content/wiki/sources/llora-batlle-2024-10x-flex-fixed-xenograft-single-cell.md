# 10x Flex for Xenograft Single-Cell Transcriptomics

## Summary

Demonstrates that 10x Gene Expression Flex protocol can simultaneously profile human and mouse cells in PDX/CDX xenograft models by mixing species-specific probe sets. Develops a pipeline to remove cross-reactive probe signal, enabling species-specific count tables without flow sorting. Compatible with fixed and frozen samples.

## Key points

- Mixed human+mouse probe sets enable dual-species scRNA-seq
- Cross-reactivity removal pipeline for clean species separation
- No flow sorting required for human/mouse cell deconvolution
- FFPE and fresh-frozen compatible

## Project relevance

- Practical method for xenograft single-cell analysis
- Enables simultaneous tumor and microenvironment profiling
- Applicable to preclinical drug testing models

## Caveats

- Preprint; not peer-reviewed
- Probe-based, not whole-transcriptome
- Some cross-reactivity may persist
