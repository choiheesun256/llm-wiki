# Orbitrap Astral Phosphoproteomics

## Summary

Demonstrates the Orbitrap Astral mass spectrometer with DIA for rapid, deep phosphoproteome analysis: ~30,000 phosphosites in 30 minutes. Generates mouse multi-tissue phosphoproteome atlas (81,120 sites across 12 tissues in 12 hours). Integrates AlphaFold structure predictions to examine structural context of phosphorylation. Benchmarked against synthetic standards and prior MS platforms.

## Key points

- ~30,000 phosphosites in single 30-min DIA run
- 81,120 unique sites in mouse 12-tissue atlas
- Outperforms Orbitrap Ascend and timsTOF platforms
- AlphaFold integration for structural phosphosite context

## Project relevance

- Next-generation phosphoproteomics technology benchmark
- DIA workflow for high-throughput phosphorylation analysis
- Multi-tissue phosphoproteome atlas resource

## Caveats

- New instrument with limited availability
- Mouse atlas; human phosphoproteome differences expected
- DIA analysis software still evolving
- Site-level quantitation accuracy limitations
