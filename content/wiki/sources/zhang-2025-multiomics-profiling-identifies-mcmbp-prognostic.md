# MCMBP as Prognostic Biomarker in PAAD (Frontiers in Immunology 2025)

## Summary
Multi-omics analysis across public databases (TCGA, CPTAC, GTEx, GEO) identifies MCMBP as an independent prognostic factor in pancreatic ductal adenocarcinoma (PAAD). MCMBP overexpression activates the JAK-STAT3 pathway to upregulate PD-L1, creating an immunosuppressive TME. Low-MCMBP patients show greater sensitivity to anti-PD-L1 immunotherapy. Drug sensitivity analysis nominates Gemcitabine+Paclitaxel, Tozasertib, and Motesanib as candidate treatments for MCMBP-high PAAD.

## Key points
- Pan-cancer (TCGA, 33 types) and focused PAAD analysis (TCGA + CPTAC + 4 GEO datasets)
- MCMBP overexpression correlates with poor prognosis; Cox regression confirms independence in PAAD
- Low DNA methylation + high m6A modification promotes PAAD progression via MCMBP
- MCMBP → JAK-STAT3 activation → PD-L1 upregulation → immunosuppressive TME
- MCMBP-low patients: better anti-PD-L1 response (validated in IMvigor210 cohort)
- Drug sensitivity: GDSC + CMap identifies Gemcitabine+Paclitaxel, Tozasertib, Motesanib
- MCMBP knockdown: inhibits proliferation, migration, invasion, G1-S transition in PAAD cells

## Project relevance
- GDSC + Connectivity Map drug sensitivity pipeline is directly applicable to drug repurposing work
- JAK-STAT3-PD-L1 immune suppression axis as a drug target framework
- TIDE-based immunotherapy response prediction methodology is a standard reference
- Demonstrates how public multi-omics databases can be combined for biomarker discovery without new patient data

## Caveats
- Primarily bioinformatics-based; no prospective clinical validation
- Functional validation limited to cell lines; no in vivo models
- Single institution study (Xi'an Medical University)
- Pan-cancer conclusions should be interpreted with PAAD-specific context in mind
