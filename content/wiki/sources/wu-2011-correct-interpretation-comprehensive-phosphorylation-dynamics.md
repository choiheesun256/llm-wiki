# Phosphoproteomics Normalization by Protein Expression

## Summary
Demonstrates that phosphoproteomics data must be normalized to protein expression changes for correct biological interpretation. Using yeast MAPK signaling as model, shows that ~30% of phosphorylation changes are explained by protein abundance changes rather than kinase/phosphatase regulation. Proposes a normalization workflow for comprehensive phosphorylation dynamics studies.

## Key points
- Phosphorylation changes must be normalized to protein abundance for accurate interpretation
- ~30% of observed phospho changes are protein-driven rather than regulation-driven
- Yeast pheromone MAPK pathway used as model system
- Essential methodology for all comprehensive phosphoproteomics studies

## Project relevance
- Core methodology for phosphoproteomics normalization
- Applicable to CPTAC-style integrated analyses
- Foundation for interpreting kinase activity from phospho data

## Caveats
- Yeast model system (2011 study)
- SILAC-based quantification (different from TMT/label-free)
- Conceptually simple but often overlooked in practice
