# Multi-omics Drug Discovery Review

> 📄 [*PMC* (2024)](https://pmc.ncbi.nlm.nih.gov/articles/PMC11954193/)

## 요약

2015-2024년간 발표된 network 기반 multi-omics 통합 방법을 체계적으로 리뷰한 논문이다. Drug target 식별, drug response 예측, drug repurposing에서의 응용을 정리한다.

## 핵심 내용

- 네 가지 주요 접근법으로 분류: network propagation/diffusion, similarity 기반, GNN, network inference
- 각 접근법의 장단점과 적용 가능 시나리오를 비교
- Multi-omics 데이터 (genomics, transcriptomics, proteomics, metabolomics)의 네트워크 통합이 단일 omics 대비 우수

## Drug repurposing과의 관련성

- 이 프로젝트에서 활용할 수 있는 computational 방법론의 전체 지도를 제공
- Network propagation은 disease module 발견에, GNN은 KG 기반 repurposing에 활용
- 데이터 통합 전략 선택의 가이드라인 역할
